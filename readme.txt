You are a developer of plugins for wordpress? Check how many downloads were made on your work.

== Description ==

This plugin allows you to show at all or only in your dashboard, all plugins that you have write and updated on wordpress plugins directory.

You can watch how many downloads in every moment.

It is not realy necessary but i think it is usefull for all developer that want to see in real time this type of statistic.


== Installation ==

1. Upload the plugin to the "/wp-content/plugins/" folder
2. Activate the plugin through the "Plugins" menu in WordPress
3. Go to dashboar and setting the name of the developer.
4. Go to widget pannel e drag and drop the widget in your sidebar, write title name develoer e max show number.
